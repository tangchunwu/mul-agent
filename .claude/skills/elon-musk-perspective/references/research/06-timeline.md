# 06 人物时间线

## 关键时间线

| 时间 | 事件 | 对思维方式的影响 |
|------|------|------------------|
| 1971-06-28 | 出生于南非比勒陀利亚 | 其后形成强烈的工程、科幻与系统控制倾向 |
| 1995 | 参与创办 Zip2 | 从软件与互联网创业起步 |
| 1999 | X.com 创建 | 金融系统与网络平台经验积累 |
| 2002-05 | 创办 SpaceX | 多行星目标与硬核工程路线开始成形 |
| 2004-04 | 加入 Tesla 董事会 | 电动车与能源系统方向加深 |
| 2008-10 | 任 Tesla CEO | 从投资者/董事转向直接经营者 |
| 2015 | 联合创办 OpenAI（后离开） | 对通用 AI 的长期关注公开化 |
| 2016 | Neuralink、The Boring Company 等项目继续扩展 | 其“并行多前线推进”风格成形 |
| 2022-10 | 收购 Twitter（后更名 X） | 其影响力从工业系统扩展到信息平台 |
| 2023 | xAI 创立，Grok 发布 | AI 进入其主战场之一 |
| 2025-01-29 | Tesla 发布 2024 财报 | 继续推进自动驾驶、能源与制造主线 |
| 2025 | Tesla Robotaxi 在美国得州奥斯汀有限区域运营 | 自动驾驶服务首次以官方支持页形式落地 |
| 2026-02-02 | xAI 官方宣布加入 SpaceX | 算力、航天与 AI 进一步整合 |
