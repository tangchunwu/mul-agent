# 06 人物时间线

## 关键时间线

| 时间 | 事件 | 对思维方式的影响 |
|------|------|------------------|
| 1958 | 出生 | 其后成长为兼具图形设计、展览策划与设计教育背景的跨域设计者 |
| 1983 | 毕业于武藏野美术大学设计专业 | 建立其与设计教育、视觉系统和社会观察的长期连接 |
| 1991 | 成立 Hara Design Institute | 开始形成独立的方法论与设计团队 |
| 1998 | 参与长野冬奥会开闭幕式项目 | 将文化叙事与视觉设计结合 |
| 2000 | 策划 `RE-DESIGN: Daily Products of the 21st Century` | 明确提出“日常中蕴含巨大设计资源”这一核心判断 |
| 2002 | 进入 MUJI 顾问委员会并开始担任艺术总监 | 将其哲学长期写入商业品牌与生活系统 |
| 2004 | 策划 `HAPTIC - Awakening the Senses` | 强化“感官即设计资源”的方向 |
| 2007 / 2009 | 组织 `TOKYO FIBER -- SENSEWARE` | 继续以展览形式呈现材料与工业潜力 |
| 2008-2009 | 举办 `JAPAN CAR` 展 | 把产业潜力与文化表达结合 |
| 2015 | 出版 `Ex-formation` 英文版 | 将“未知化”明确化为方法论概念 |
| 2016 | 主导 JAPAN HOUSE 创意方向 | 把“空的容器”逻辑用于国家形象表达 |
| 2019 | 出版 `Designing Japan` 与 `100 Whites` | 更系统地把日本的美学资源与未来联系起来 |
| 2022 | 推出 `SUWARI` 系列 | 继续把空间感、身体感与日常家具统一起来 |
| 2024 | `DRAW` 项目发布 | 将素描与“把不稳定念头拖入现实”的方法公开化 |
| 2025 | MUJI `Like Water, Like Air` 与 ReMUJI 传播项目 | 继续把品牌视为生活气候与循环系统的一部分 |

## 对蒸馏的影响

- 原研哉并非历史人物，其方法仍在继续演进。
- 因此这版 skill 可以适度纳入近年的 MUJI、ReMUJI、DRAW 等项目，但不能把近年商业传播误缩减为他的全部思想。
