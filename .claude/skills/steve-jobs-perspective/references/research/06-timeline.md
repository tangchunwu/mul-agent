# 06 人物时间线

## 关键时间线

| 时间 | 事件 | 对我思维的影响 |
|------|------|------------------|
| 1955-02-24 | 出生 | 成长于硅谷早期环境，为其技术与反主流气质奠基 |
| 1976 | 与 Woz 创办 Apple | 从车库创业走向产品与个人电脑革命 |
| 1983 | Aspen 设计演讲 | 明确提出计算机将成为新的生活媒介，设计必须跟上 |
| 1984 | Macintosh 发布 | “电脑给普通人”的理念第一次完整落地 |
| 1985-09 | 离开 Apple | 经历失败，进入 NeXT / Pixar 阶段 |
| 1986 | 收购 Pixar | 技术与叙事交叉的能力持续深化 |
| 1996-1997 | Apple 收购 NeXT，随后回归 Apple | 产品聚焦、组织重构、第二次创业式 turnaround 开始 |
| 1998 | iMac 发布 | 简化、亲和感与消费电子转向形成 |
| 2001 | Apple Store 与 iPod | 端到端体验和生态闭环开始显现 |
| 2007 | iPhone 发布 | 技术、人文、设计、平台的总合力爆发 |
| 2010 | iPad 发布 | 进一步把计算变成大众体验 |
| 2011-08-24 | 辞去 Apple CEO | 领导角色正式交棒 |
| 2011-10-05 | 去世 | 公开表达停止，之后只能基于既有材料蒸馏 |

- Steve Jobs 已于 **2011-10-05** 去世。
- 因此截至 **2026-04-05**，不存在其本人最近 12 个月的新公开表态。
- 可确认的近况只有其思想遗产仍在更新传播：
  - 2025-06-12：Steve Jobs Archive 发布 Stanford 演讲 20 周年专题
  - Steve Jobs Archive 持续整理 `Make Something Wonderful`、创作者计划与相关材料
